# Premium product rules

Design and build like the teams behind **Stripe, Linear, and Vercel**.

## Design principles
- **Avoid generic AI aesthetics.** Intentional, distinctive choices — no templated defaults.
- Priority order of craft:
  - **Typography** — deliberate scale, weight, rhythm.
  - **Spacing** — consistent, generous, grid-aligned.
  - **Hierarchy** — clear visual priority; guide the eye.
  - **Motion** — purposeful animation (GSAP), respect `prefers-reduced-motion`.
  - **Accessibility** — semantic HTML, keyboard nav, contrast, focus states.
  - **Performance** — fast first paint, small bundle, no layout shift.
